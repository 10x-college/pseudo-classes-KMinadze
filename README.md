­ЪЊї рЃљрЃдрЃгрЃћрЃарЃљ

рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃљ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃљ HTML рЃЊрЃљ CSS рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.
рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃАрЃБрЃцрЃЌрЃљ, рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃў рЃЊрЃљ рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃњрЃљрЃўрЃЏрЃћрЃЮрЃарЃћрЃЉрЃА Figma-рЃерЃў рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃЊрЃўрЃќрЃљрЃўрЃюрЃА.

2. Selector Specificity

рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў (class).

рЃЏрЃЮрЃћрЃарЃўрЃЊрЃћрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃбрЃћрЃњрЃћрЃЉрЃўрЃЌ рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА (рЃЏрЃљрЃњ: div p, section h1).

рЃЏрЃЮрЃћрЃарЃўрЃЊрЃћрЃЌ ID-рЃћрЃЉрЃўрЃЌ рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА, рЃарЃљрЃЌрЃљ рЃЎрЃЮрЃЊрЃў рЃўрЃДрЃЮрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃЊрЃў рЃЊрЃљ рЃњрЃљрЃАрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃћрЃџрЃў.

3. Unique IDs

рЃДрЃЋрЃћрЃџрЃљ ID рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў.

рЃћрЃарЃЌ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃЊрЃћрЃА рЃЮрЃарЃў рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃў ID.

рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃбрЃћрЃњрЃћрЃЉрЃўрЃЌ (p, h1, div) рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў.

рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ ID-рЃћрЃЉрЃў, рЃЌрЃБрЃЏрЃфрЃљ рЃћрЃА рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃљрЃа рЃљрЃарЃўрЃА.
